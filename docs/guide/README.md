# Food Porter — User Guide

Screen-by-screen documentation of the Food Porter web app for **every persona** — donors,
volunteers, drivers, and foundation staff (admins) — plus the public homepage.

## What's here

- **[HANDBOOK.md](HANDBOOK.md)** — the complete handbook. One layered book: each persona's
  section is written for that persona, and the admin section doubles as the staff training +
  support reference. Every screen is shown with a real screenshot.
- **[assets/](assets/)** — the screenshots (captured from the live application, English UI).
- **`FoodPorter-Handbook.html`** *(generated)* — a self-contained, printable version of the
  handbook for sharing with the foundation team. Open it in any browser; no internet needed.

## How to read it

- **New donor/volunteer/driver?** Read section 1 (For everyone) then your own section.
- **Foundation staff / trainer?** Read the whole thing — the Admin section (6) is the operational
  manual; the persona sections tell you what each user sees so you can support them.

## Keeping it current

The screenshots are from the running app. When a screen changes materially, re-capture that one
image into `assets/` (same filename) and, if the flow changed, update its section in
`HANDBOOK.md`. The handbook is versioned in git so edits are reviewed like any other change.

## A note on language

All screenshots show the **English** interface. Every screen also exists in **Gujarati** and
**Hindi** — users switch with the EN · ગુ · हि toggle in the top bar, and the same layouts apply.
