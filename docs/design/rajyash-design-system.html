Spacing, radius & elevation

A 4px base scale keeps rhythm consistent and Tailwind-aligned. Corners are friendly but not bubbly (12px base). Shadows are warm-tinted and soft — never harsh — to keep the interface calm and trustworthy.

Status system

Pickup status pills

Each state pairs a distinct hue, a coloured dot, and a label — never colour alone — so status is legible for colour-blind users and in sunlight. The same tokens drive cards, tables, timelines and toasts.

Multilingual-ready

The same component renders in English, ગુજરાતી and हिन्दी. Strings live in data — never baked into images — and layouts flex for text that can grow 30–40% longer. Generous line-height keeps Indic scripts readable.

1 · You offer food

Restaurants, hosts and families post surplus food in under a minute.

2 · A driver picks up
