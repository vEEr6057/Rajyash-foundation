| Developer workstation → git | .dev.vars and .env must never be committed; enforced via .gitignore |
| CI → CF deploy | Secrets injected as Cloudflare Worker env vars, not baked into build |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Information Disclosure | .dev.vars / .env | mitigate | Add `.dev.vars` and `.env` to .gitignore; only `.env.example` and `.dev.vars.example` committed |
| T-01-02 | Information Disclosure | package.json / lockfile | accept | No secrets in package.json; versions public by design |
| T-01-03 | Tampering | wrangler.jsonc compatibility_date | mitigate | Pin `2025-05-05` — do not lower; lower date breaks postgres-js (FinalizationRegistry) |
</threat_model>
